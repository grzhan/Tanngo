Tanngo
======

一个用来学习日语的简单HTML5移动app，目前是为了提交作业而完成的原型版本

本App是基于UI框架ionic开发（AngularJS + Phonegap），目前提供了基础的单词本功能和语音识别功能
语音识别由服务器端实现（ffmpeg转码+Julius识别），目前还不成熟【就糊弄下老师
