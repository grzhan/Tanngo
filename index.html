Hi, I´m grzhan, thanks for using Tanngo

Tanngo is a simple mobile app which can help you learn Japanese kana(words),based on ionic , a HTML5 mobile UI framework.

The tanngo´s most amazing feature is kana recognization(in Accent Test), tanngo can record your pronounciations, then analyze and score them. What help us implements this feature is Julius, a text by alarge vocabulary continuous speech recognizer(LVCSR), we deploy it in a external web server. The mechanism is Julius receive audio datas from tanngo and send back results to mobile-end.

Tanngo is still in Beta, so if you find any bugs or you have any suggestions, please click button below to contact us.
